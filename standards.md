# Standard guide

## Commits

### Options
- chore - administrative work (markdown files, gitignore, merge conflicts)
- style - css, styling, formatting
- feat - features
- fix - bug fixes
- docs - documentation

### Layout

Use the following layout to make a new commit.

`option(filename): description`

Example:

`feat(view.ts): added button for return-funtionality`

## Branches

### Options
- chore - administrative work 
- style - css, styling, formatting
- feat - features
- fix - bug fixes
- docs - documentation

### Layout

Use the following layout to make a new branch.

`option/feature`

Example:

`feat/back-button`
